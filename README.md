# IrrigationSystem
Raspberry Pi project to simulate an hourly irrigation system using Python, DHT sensor and data from the CIMIS website

Hardware Requirements:
 - Temperature and Humidity Sensor (DHT-11)
 - PIR Sensor
 - LCD
 - Relay
 - Motor
